# CampusSN — Pages légales

Site statique hébergeant les documents légaux publics de l'application CampusSN,
servis via **GitHub Pages**.

- `index.html` — page d'accueil avec les liens
- `privacy.html` — Politique de confidentialité
- `terms.html` — Conditions d'utilisation (CGU)

> `.nojekyll` désactive le traitement Jekyll : les fichiers HTML sont servis tels quels.

Les URLs publiques sont renseignées dans l'app (`src/config/contact.js` :
`PRIVACY_URL`, `TERMS_URL`) et dans les fiches App Store / Play Store.
